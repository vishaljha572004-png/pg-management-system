- # 🏠 RentWeb - Multi-Tenant PG Management System

> A Production-Ready Multi-Tenant PG Management SaaS Platform built with React, Node.js, Express.js, and MySQL.

RentWeb is a modern PG (Paying Guest) Management System designed to simplify hostel and PG operations. It allows multiple PG Owners to manage their properties independently through a single platform while ensuring complete tenant isolation, secure authentication, and real-time management of rooms, students, payments, complaints, and notifications.

# 🚀 About the Project

RentWeb is a commercial SaaS-ready PG Management System that provides a complete solution for PG Owners, Students, and Platform Administrators.

The platform supports:

- Multi-Tenant Architecture
- Separate Dashboard for each PG
- Role-Based Authentication
- Room Management
- Rent Management
- Electricity Billing
- QR Based Payments
- Complaint Management
- Notification System
- Profile Verification
- Police Verification
- Receipt Generation

The application is designed to be scalable, secure, and production-ready.

# 🏢 Multi-Tenant SaaS Architecture

The application supports multiple PGs using a single codebase.

Every PG has:

- Separate Admin
- Separate Students
- Separate Rooms
- Separate Payments
- Separate QR Code
- Separate Notices
- Separate Reports

No data is shared across PGs.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/RentWeb.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

DB_HOST=localhost

DB_PORT=3306

DB_NAME=rentweb

DB_USER=root

DB_PASSWORD=your_password

JWT_SECRET=your_secret_key
```
